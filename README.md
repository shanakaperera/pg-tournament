# pg-tournament
Pokerguild Tournament Application

## Installation

Clone the repo locally:

```sh
git clone https://github.com/shanakaperera/pg-tournament.git pg-tournament
cd pg-tournament
```

Install PHP dependencies:

```sh
composer install
```

Install NPM dependencies:

```sh
npm install
```

Build assets:

```sh
npm run dev
```

Setup configuration:

```sh
cp .env.example .env
```

Generate application key:

```sh
php artisan key:generate
```
